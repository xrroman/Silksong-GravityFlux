# 🌌 GravityFlux - Silksong Mod

**GravityFlux** is a utility for _Hollow Knight: Silksong_ that grants you absolute control over the laws of physics in Pharloom. From precise gravity adjustments to a chaotic "Random Mode," this mod fundamentally changes how Hornet interacts with the world.

---

## 🚀 Features

- **Real-Time Control Panel:** An integrated UI to adjust gravity settings on the fly without pausing.
- **Precise Increments:** Increase or decrease the gravity scale in increments of **0.5x**.
- **Chaos Mode (Random Gravity):** Automatically shifts gravity to a random value (between **0.2x** and **3.0x**) at set intervals.
- **Configurable Intervals:** You decide exactly how many seconds pass before the next gravity shift.
- **Instant Reset:** Quickly return to the game's default physics with a single click.

---

## 🎮 How to Use

Once installed, you can control the mod using the following inputs:

| **Action**            | **Key**                          |
| --------------------- | -------------------------------- |
| **Open / Close Menu** | `F3`                             |
| **Quick Close**       | `ESC` (While the menu is active) |
| **Move Panel**        | Click and drag the title bar     |

### Menu Instructions:

1. **Scale:** Displays the current gravity multiplier applied to Hornet.
2. **Increase/Decrease:** Modifies Hornet's weight. Low gravity allows for massive jumps and slow descents; high gravity makes you drop like a stone.
3. **Random Mode:** Activates the countdown timer. The panel will display the time remaining until the next physical shift.

---

## 🛠 Installation

1. Ensure you have **BepInEx** installed for Silksong.
2. Download the `GravityFlux.dll` (or install directly via the Thunderstore manager).
3. Place the file into your `Silksong/BepInEx/plugins` folder.
4. Launch the game and press `F3`!

---

## 📝 Technical Overview

The mod utilizes a dynamic scaling system to ensure that Hornet's movement remains fluid and responsive, regardless of the gravity level. Unlike simple physics overrides, this multi-variable approach ensures that jump trajectories, terminal velocity, and aerial momentum all scale in perfect harmony, preserving the tight "feel" of Silksong's gameplay under any gravitational condition.
